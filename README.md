# 🍔 MFD - Mohit Food Delivery

MFD (Mohit Food Delivery) is a **full-stack food delivery web application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

The project includes a **customer website**, an **admin dashboard**, and a **backend server** for managing food items, categories, orders, and users.

---

# 🚀 Project Structure

The project is divided into three main parts:

MFD
│
├── server → Backend API (Node.js + Express + MongoDB)
├── food → Customer website (React)
└── dashboard → Admin panel (React)


---

# 🛠 Tech Stack

Frontend:
- React.js
- Axios
- Bootstrap / CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
- Multer (Image Upload)
- Bcrypt (Authentication)

Tools:
- Git & GitHub
- VS Code

# ✨ Features

### Customer Website
- View food menu
- Browse food categories
- View food details

- Responsive UI

### Admin Dashboard
- Admin login authentication
- Manage categories
- Manage food items
- Manage blogs
- Manage customers
- Upload images

### Backend
- REST API
- MongoDB database

# ⚙️ Installation

1.Clone the repository

git clone https://github.com/Mohitthakur123/MFD-Food-Delivery.git

cd MFD-Food-Delivery


---

# ▶️ Run Backend (Server)
cd server
npm install
npm start
Server runs on:
http://localhost:5000


---

# ▶️ Run Customer Website
cd food
npm install
npm start
runs on:
http://localhost:3000

---

# ▶️ Run Admin Dashboard
cd dashboard
npm install
npm start
runs on:
http://localhost:2000
